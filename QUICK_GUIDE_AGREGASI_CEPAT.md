# Quick Guide: Agregasi Cepat

## ⚡ 3 Langkah Mudah

### 1️⃣ Muat Data
```
Buka Data Builder → Pilih kolom → Klik "Tampilkan Data"
```

### 2️⃣ Lihat Agregasi
```
Klik tab "Agregasi Cepat"
```

### 3️⃣ Export
```
Klik "Export Excel"
```

## 📊 Apa yang Ditampilkan?

### Pangkat Utama
```
I, II, III, IV, PPPK, Non ASN
(tanpa a/b/c/d)
```

### Pendidikan
```
S3, S2, S1, D4, D3, SMA/SMK
(tanpa jurusan)
```

### Jenis Kelamin
```
Laki-laki, Perempuan
```

## 💡 Tips

✅ **Untuk Laporan Cepat**: Gunakan tab "Agregasi Cepat"
✅ **Untuk Analisis Detail**: Gunakan tab "Statistik"

## 📁 Output Excel

4 sheet sederhana:
1. Ringkasan
2. Pangkat Utama
3. Pendidikan
4. Jenis Kelamin

## 🎯 Contoh Penggunaan

### Laporan Bulanan
```
1. Pilih kolom: Pangkat/Golongan, Gender
2. Pilih Data Relasi: Riwayat Pendidikan
3. Tampilkan Data
4. Tab "Agregasi Cepat"
5. Export Excel
6. Kirim ke pimpinan ✅
```

### Perbandingan Bulan
```
Maret: Export → agregasi-cepat-2026-03-31.xlsx
April: Export → agregasi-cepat-2026-04-30.xlsx
Bandingkan angka-angka ✅
```

## ⚠️ Catatan

- Pastikan pilih **"Riwayat Pendidikan"** di Data Relasi untuk agregasi pendidikan
- Pastikan ada field **"Pangkat/Golongan"** untuk agregasi pangkat
- Pastikan ada field **"Gender"** untuk agregasi jenis kelamin

## 🚀 Mulai Sekarang!

Buka Data Builder → Tab "Agregasi Cepat" → Export!
